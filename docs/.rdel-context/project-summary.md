# RDEL Project Summary

## Project

Contollo.Rdel.ZipRunner

## Purpose

Visual Studio 2022 extension and RDEL runner for applying portable AI-assisted change packages.

## Current Strategic Role

RDEL is the change/package engine inside the broader Neuro Commander Studio vision.

## Core Principle

The prompt is not the product. Context is the product.

RDEL packages carry change state.

RDEL Session Protocol carries AI session state.

Context packages carry portable AI context state.

## Current Major Systems

- RDEL package apply/test/rollback
- Git checkpointing
- package verification/history
- AI Session / RDEL Session Protocol
- Context Assembly Engine
- context levels
- context package export
- settings UI
- output window capture
