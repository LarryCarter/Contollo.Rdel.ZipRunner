# Reusable Pattern: Convert AI Chat Decisions into Engineering Memory

## Problem

Important engineering reasoning often happens in AI chat, but later disappears into long chat history.

## Pattern

When a chat produces a non-obvious decision, fix, lesson, rejected approach, or reusable gotcha, promote it into Engineering Intelligence.

## Capture

Record:

- problem
- context
- attempts
- failed approaches
- final fix
- reason
- related files
- related packages
- related commits
- reusable lesson
- future warning
